# 2020-02-05-learn

## nginx

```cmd
yum install nginx

#设置开机启动
$ systemctl enable nginx

#启动服务
$ systemctl start nginx

#停止服务
$ systemctl restart nginx

#重新加载，因为一般重新配置之后，不希望重启服务，这时可以使用重新加载。
$ systemctl reload nginx

#修改配置文件
vi /etc/nginx/nginx.conf
# i 修改 esc退出 :wq 保存

# 查看端口
netstatus 
```
