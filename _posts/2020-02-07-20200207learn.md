### 解决 matplotlib 的中文乱码问题

首先引入 matplotlib，然后借助 matplotlib.matplotlib_fname() 求出需要修改的文件地址。

然后在将 matplotlibrc 文件中的
```
#font famaly: sans-serif
```
修改为
```
font family: Microsoft YaHei
```

原理：默认字体无法识别中文，故修改 matplotlib 渲染时的默认字体。
